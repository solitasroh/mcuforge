01 — Executive Summary

프로젝트 개요

embtool은 NXP Kinetis 및 STM32 MCU를 대상으로 ARM GCC 툴체인의 설치·관리·전환과 프로젝트 스캐폴딩을 자동화하는 CLI 도구입니다.

목적
현재 수동으로 이루어지는 임베디드 개발 환경 구축 과정을 자동화하여, 개발자 온보딩 시간을 반나절에서 5분으로 단축하고, 팀 전원의 빌드 환경을 동일하게 유지합니다.
대상 사용자
주 사용자: 4-5명 규모의 MCU 펌웨어 개발팀
개발 환경: Windows 11 (IDE: CLion / VS Code)
CI/CD: GitLab CI Runner 또는 Jenkins Agent (Linux)
핵심 가치
1. 툴체인을 프로젝트 밖에서 관리 (nvm/rustup 방식)
2. embtool.toml 하나로 팀 환경 동기화
3. 한 줄 명령으로 프로젝트 생성·빌드·플래시
기술 스택
언어: Rust 1.94 (단일 바이너리 배포)
빌드 시스템: CMake 래핑
디버거: PEMicro Multilink (SWD)
배포: GitHub Releases (Win/Linux)

05 — 상세 명령어 설계

CLI 명령어 체계

모든 명령어는 embtool <verb> [options] 패턴을 따릅니다.

주요 명령어 상세

embtool setup — 환경 자동 구성 Phase 0

프로젝트의 embtool.toml을 읽고, 필요한 툴체인을 자동 설치하며, arm-toolchain.cmake를 (재)생성합니다. --ci 플래그로 CI 환경에서 대화형 프롬프트를 비활성화합니다.

옵션설명기본값
--ciCI 환경 모드 (비대화형)자동 감지
--force기존 설정 덮어쓰기false
--mirror <url>FTP 미러 URL 임시 지정config.toml

embtool toolchain install <version> — 툴체인 설치 Phase 1

ARM GNU Toolchain을 사내 FTP 미러 (우선) 또는 ARM 공식 사이트 (fallback)에서 다운로드하여 ~/.embtool/toolchains/에 설치합니다. Windows에서는 .zip, Linux에서는 .tar.xz를 자동 감지합니다.

명령설명
toolchain install 13.3ARM GCC 13.3.rel1 설치
toolchain list설치된 버전 목록 (active 표시)
toolchain use 12.2전역 기본 버전 전환
toolchain remove 10.3지정 버전 삭제

embtool new <name> --mcu <mcu> — 프로젝트 생성 Phase 2

MCU 데이터베이스에서 타겟 정보를 조회하고, CMakeLists.txt, arm-toolchain.cmake, embtool.toml, startup 코드, 링커 스크립트를 자동 생성합니다.

옵션설명기본값
--mcu <alias>타겟 MCU (k10d, k22f, k64, k66)필수
--type <type>application / bootloader / libraryapplication
--toolchain <ver>사용할 툴체인 버전기본 active

embtool build / embtool flash — 빌드 & 플래시 Phase 3

build는 CMake를 래핑하여 빌드를 수행하고, Flash/RAM 사용량을 자동 리포트합니다. flash는 PEMicro 프로브를 자동 감지하여 .elf를 플래시합니다.

옵션설명기본값
--release / --debug빌드 프로파일debug
--clean빌드 디렉토리 정리 후 빌드false
--verboseCMake 상세 출력false

embtool migrate — 레거시 마이그레이션 Phase 4

기존 CMakeLists.txt와 arm-toolchain.cmake를 분석하여 embtool.toml을 자동 생성하고, arm-toolchain.cmake를 embtool 관리 버전으로 교체합니다. 기존 파일은 .bak으로 백업됩니다.

07 — 설정 파일 설계

Configuration

프로젝트 설정은 embtool.toml에, 전역/개인 설정은 ~/.embtool/config.toml에 관리합니다.

embtool.tomlGit 커밋
[project]
name = "a2750lm_application"
version = "1.0.0"

[target]
mcu = "MK64FN1M0VLL12"
core = "cortex-m4"
fpu = "soft"
flash = "1M"
ram = "256K"

[toolchain]
version = "13.3.rel1"

[build]
c_standard = "c99"
optimization.debug = "O0"
optimization.release = "O1"
linker_script = "system/linkerscript.ld"

[build.defines]
target = ["MK64F12"]

[build.flags]
common = [
  "-ffunction-sections",
  "-fno-common",
  "-fdata-sections",
  "-fsigned-char",
]

[debug]
probe = "pemicro"
interface = "swd"
~/.embtool/config.toml로컬
[toolchain]
default = "13.3.rel1"
install_dir = "~/.embtool/toolchains"

[toolchain.sources]
arm_gnu = "https://developer.arm.com/..."

[toolchain.mirror]
enabled = true
url = "ftp://192.168.x.x/toolchains/"
fallback = true

[debug]
default_probe = "pemicro"

[ci]
auto_detect = true
# GITLAB_CI, JENKINS_URL,
# CI 환경변수 자동 감지

예상 리스크 및 대응
