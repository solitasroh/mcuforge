# 이슈 템플릿

## 제목 컨벤션

```
[제품코드] [카테고리] 간결한 설명
```

### 제품코드

| 코드 | 설명 |
|------|------|
| A2700 | A2700 시스템 전체 |
| A2750 | A2750 IRM 모듈 |
| A2750P | A2750P 변형 |
| AccuraLogic | AccuraLogic 소프트웨어 |

### 카테고리

| 코드 | 설명 |
|------|------|
| ADC | ADC/DMA/측정 관련 |
| HW | 하드웨어 인터페이스 |
| FW | 펌웨어 일반 |
| Modbus | 통신 프로토콜 |
| UI | 사용자 인터페이스 |
| Build | 빌드 시스템 |
| Doc | 문서화 |
| Test | 테스트 |

### 예시

```
[A2750] [ADC] DMA 전송 시 데이터 누락 발생
[A2750] [FW] HardFault 발생 시 복구 로직 추가
[A2700] [Modbus] 레지스터 맵 확장 (#32000~#32050)
```

## 본문 템플릿

### Bug

```markdown
## 현상
- 어떤 조건에서 어떤 문제가 발생하는지

## 재현 절차
1. ...
2. ...

## 예상 동작
- 정상적으로 동작해야 하는 방식

## 실제 동작
- 현재 발생하는 문제

## 환경
- FW 버전:
- HW 리비전:
```

### Feature

```markdown
## 목적
- 이 기능이 필요한 이유

## 요구사항
- [ ] 기능 1
- [ ] 기능 2

## 설계 메모
- 구현 방향 또는 참고사항

## 테스트 계획
- 검증 방법
```

### Task

```markdown
## 목적
- 이 작업의 목적

## 작업 항목
- [ ] 항목 1
- [ ] 항목 2

## 완료 조건
- 이 작업이 완료되었다고 판단하는 기준
```
